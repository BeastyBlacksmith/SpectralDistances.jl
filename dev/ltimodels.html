Models and root manipulations

Models and root manipulations

Overview

This package supports two kind of LTI models, AR and ARMA. AR represents a model with only poles whereas ARMA has zeros as well.

To fit a model to data, one first has to specify a FitMethod, the options are

LS
PLR
TLS

For example, to estimate an AR model of order 30 using least-squares, we can do the following

julia> data = randn(1000);

julia> fitmethod = LS(na=30)
LS(30, 0.01)
